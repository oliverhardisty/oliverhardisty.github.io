Filtering & Sorting
project type
New functionality for the ClearScore app
project company
ClearScore
project context
The ClearScore app has a marketplace of typical credit products including from credit cards, loans, car insurance and more. Within these products, the users only see suitable products for them based on their credit data. Despite this, the users still see several products per search, which can make it difficult for them to find the one most suitable. This project introduced filtering and sorting, to allow users to refine products by key characteristics.
design process
The process was largely agile, involving these key stages;
- Competitor analysis and market research
- UX/UI explorations and iterations
- User testing of prototypes
- Data analysis on released experiments
- Further design iterations dependent on analytics
platforms
iOS, Android and Web
approx. time frame
08-12.2019
